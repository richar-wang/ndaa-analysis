SEC. 11330.  ENHANCEMENT OF SEAFOOD IMPORT MONITORING PROGRAM MESSAGE SET IN AUTOMATED COMMERCIAL ENVIRONMENT SYSTEM.   The Deadline. Strategy.  Secretary, in coordination with the Commissioner of U.S. Customs and Border Protection, shall, not later than 6 months after the date of enactment of this Act, develop a strategy to improve the quality and verifiability of already collected Seafood Import Monitoring Program Message Set data elements in the Automated Commercial Environment system. Such strategy shall prioritize the use of enumerated data types, such as checkboxes, dropdown menus, or radio buttons, and any additional elements the Administrator of the National Oceanic and Atmospheric Administration finds appropriate.
